# docker-nginx-express

> A playground for Docker with Nginx and Express.

# Development

* Cloning the repo

```bash
$ git clone https://github.com/quangngoc430/docker-nginx-express.git
```

* Installing dependencies

```bash
$ npm install
```

# Docker

* Building an image

```bash
$ docker-compose build
```

* Running containers

```bash
$ docker-compose up
```

* Stopping  containers

```bash
$ docker-compose down
```
